# Incident Response Practice

## Objective
Simulated incident investigation and response exercises to develop analytical and mitigation skills.

## Skills Practiced
- Incident detection and analysis
- Log review and IOC identification
- Timeline reconstruction
- Containment strategies
- Documentation and reporting

## Tools Used
- SIEM simulation tools
- Linux logs
- Wireshark (network analysis)

## Notes
Practice exercises based on publicly shared labs; performed independently to understand structured incident response.
