# Digital Forensics Practice

## Objective
Hands-on exercises analyzing digital artefacts and applying basic forensic methodology.

## Skills Practiced
- Evidence identification and preservation
- File system analysis
- Hash verification and integrity checks
- Forensic reporting

## Tools Used
- Autopsy / Sleuth Kit
- Linux CLI
- Hashing utilities (md5sum, sha256sum)

## Notes
These exercises were performed independently to gain familiarity with forensic investigation techniques.
